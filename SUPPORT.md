# Support

## Getting Help

Thank you for using the eBot CS2 Uninstaller Script! Here's how to get help if you need it.

### Before Asking for Help

1. **Read the README:** The [README.md](README.md) contains comprehensive documentation about the script's functionality, options, and warnings.

2. **Run in Dry-Run Mode:** Always test with `--dry-run` first to understand what the script will do:
   ```bash
   sudo ./uninstall_ebot.sh --dry-run
   ```

3. **Check the Logs:** Review the log file for detailed information about what happened:
   ```bash
   sudo less /var/log/uninstall_ebot_cs2.log
   ```

4. **Search Existing Issues:** Check [GitHub Issues](https://github.com/ranas-mukminov/uninstall_ebot_CS2_server/issues) to see if your question has already been answered.
